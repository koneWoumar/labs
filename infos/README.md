# CONFIGURATION OF THE PROFILE

To configure the profile, do :

/labs/script/set_profile.sh

This will create the specific profile file /labs/config/labs-profile and lie it to the the user profile file.
This script will also create the next script:
- /labs/config/profile/labs-env         # for defining environment variables
- /labs/config/profile/labs-alias       # for defining alias
- /labs/config/profile/labs-fonctions   # for defining fonctions
